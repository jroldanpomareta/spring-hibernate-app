# 🗄️ Spring Hibernate App

Aplicación backend desarrollada con Spring y Hibernate que permite la gestión de datos mediante acceso a base de datos usando ORM.

## 🚀 Funcionalidades

* 📦 Persistencia de datos con Hibernate
* 🔄 Operaciones CRUD (crear, leer, actualizar, eliminar)
* 🗃️ Mapeo objeto-relacional (ORM)
* ⚙️ Configuración de Spring + Hibernate

## 🛠️ Tecnologías

* ☕ Java
* 🌱 Spring Framework
* 🗄️ Hibernate
* 📦 Maven

## 🧱 Estructura

* 🧩 Entidades (modelos de datos)
* 🗂️ DAO / Repositorios
* ⚙️ Configuración Hibernate
* 🔗 Integración con Spring

## ▶️ Cómo ejecutar

1. 📥 Importar el proyecto como Maven
2. ⚙️ Configurar la base de datos (si aplica)
3. 🚀 Ejecutar la aplicación
4. 📊 Probar las operaciones desde la consola o aplicación

## 🧠 Conceptos aprendidos

* Inyección de dependencias con Spring
* Integración con Hibernate
* Persistencia de datos en base de datos
* Arquitectura DAO
